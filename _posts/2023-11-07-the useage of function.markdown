函数是 Python 中的基本概念之一。函数用于将代码封装起来，以便重复使用。

**函数的定义**

函数的定义格式如下：

```python
def 函数名(参数列表):
    # 函数体
    return 返回值
```

**参数**

参数是函数需要的输入参数。参数可以是任何数据类型。

**函数体**

函数体是函数执行的代码块。函数体可以包含各种语句，包括算术运算、条件语句、循环语句等。

**返回值**

返回值是函数执行后返回的值。返回值可以是任何数据类型。函数在操作之后提供给外部的数据

**函数的调用**

函数的调用格式如下：

```python
函数名(参数列表)
```

**函数的使用**

函数可以用于以下场景：

* 将代码封装起来，以便重复使用。
* 将复杂的任务分解为多个简单的步骤。
* 将代码组织得更加清晰和易读。

**函数的例子**

```python
def add(x, y):
    """
    加法函数

    参数：
        x：第一个加数
        y：第二个加数

    返回值：
        x 和 y 的和
    """
    return x + y


print(add(1, 2))  # 3
```

**函数的注意事项**

* 函数名必须是合法的 Python 标识符。
* 参数列表中的参数可以是任意数量，也可以是空。
* 函数体中的代码可以是任意语句。
* 返回值可以是任意数据类型，也可以是空。
